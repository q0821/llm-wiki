# 加密貨幣收益工具比較

> 從 [[雙幣贏]] 概念頁抽出「與替代方案比較」表格，獨立為可持續擴充的比較頁。

## 四種工具並列

| 維度 | 活期賺幣 | 現貨網格 | [[雙幣贏]] | Deribit 自行賣選擇權 |
|------|---------|---------|-----------|-------------------|
| **本質** | 借貸（把幣借給平台或其他用戶） | 震盪套利（在區間內自動低買高賣） | 選擇權賣方的零售包裝 | 選擇權賣方的原味版 |
| **典型 APY** | 1-10% | 5-50%（含浮虧） | 20-100%+（但權利金折價 40-60%） | 公允定價，無折價 |
| **收益來源** | 借貸利息 | 震盪套利價差 | 選擇權權利金 | 選擇權權利金 |
| **方向風險** | 無（除非平台倒帳） | 中（區間外套牢） | 高（被動換幣） | 視策略 |
| **流動性** | 隨時贖回 | 隨時停止 | 鎖倉不可撤 | 限價單可撤改 |
| **定價公平性** | 公開借貸利率 | 價差即時定價 | **折價 40-60%** | 市場公允 |
| **技術門檻** | 低 | 中（選區間、停損） | 中（需懂選擇權 Greek） | 高（需自行管理倉位與擔保品） |
| **適用對象** | 穩健派、HODLer | 盤整行情偏好者 | 符合三條件的專業散戶（見下） | 有經驗的選擇權交易者 |

## 風險調整後報酬（RAR）

根據 [[src-dual-currency-truth]] 的分析：

> 「活期賺幣 + 手動限價單」的風險調整後報酬**常優於**雙幣贏，因為保留完全流動性與靈活性，且沒有折價懲罰。

這是因為雙幣贏的三個結構性劣勢：
1. **定價折價**：期望值先天就輸
2. **流動性鎖死**：鎖倉期內不可撤
3. **尾端風險**：選擇權賣方的 payoff 負偏態

相同曝險下，活期賺幣 + 限價單幾乎在所有維度都更好。

## 選擇決策樹

```
要不要承擔方向風險？
├─ 不要 → 活期賺幣
└─ 要
   ├─ 盤整行情 + 有區間觀點 → 現貨網格
   ├─ 有明確進出場價位 + 懂選擇權 + 配置 <20%
   │  ├─ 能接受定價折價與鎖倉 → 雙幣贏
   │  └─ 能自行管理倉位與擔保品 → Deribit 自賣選擇權
   └─ 其他情況 → 回到活期賺幣 + 手動限價單
```
