# 台股大跌換股操作記錄 (2026-03-09)

> 來源：Telegram 直接貼文｜記錄日期：2026-03-09

## 摘要

台股大跌日，觀察到富邦台50 跌幅較小、00981A 跌幅較大（安全邊際更高），執行換股操作：賣出富邦台50 105 張，買入 00981A 1,000 張。

## 操作明細

| 項目 | 內容 |
|------|------|
| 賣出 | 060208 富邦台50，105 張 |
| 買入 | 00981A 主動統一台股增長，1,000 張 |
| 買入價 | 18.43 |
| 比例 | 約 1:10 |
| 帳戶 | 鈴鼓帳戶 |
| 市場情緒 | 恐慌賣壓 |

## 策略邏輯

- **市場觀察**：台股大跌，富邦台50 跌幅相對小，00981A 跌幅較大
- **調整理由**：00981A 為主動管理型 ETF，具較高安全邊際，反彈潛在報酬較高
- **心理狀態**：避免情緒驅動，操作依據明確——以相對抗跌資產換取後續漲幅

## 待觀察

- 富邦台50 強弱變化
