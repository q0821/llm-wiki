# 胖哥攝影工作室 — 網站內容優化提案

> 來源：自行撰寫提案｜記錄日期：2026-03-20

## 摘要

針對[[胖哥攝影工作室]]的商業攝影網站優化提案。核心問題：430 篇文章中 67% 為婚禮相關，Google 將網站定位為「婚禮攝影師」，商業客戶搜尋時難以找到。

## 核心問題

| 問題 | 說明 |
|------|------|
| 內容偏向婚禮 | 商業活動作品佔比太低 |
| 缺商業資訊 | 企業客戶找不到能承接什麼、怎麼詢價 |
| 作品無說明 | 只有照片，無活動背景資訊 |
| 無聯繫動機 | 看完不知下一步 |

## 三層改善策略

### 第一步：讓人找得到

1. **修改副標題**（5 分鐘，立即做）
   > 現在：婚攝 / 婚禮攝影 / 婚禮記錄 / 個人寫真 / 自助婚紗 / 活動記錄
   > 建議：台北商業活動攝影｜企業形象照｜婚禮紀錄 — 胖哥攝影工作室

2. **撰寫商業攝影知識文章**（最重要，每篇 2-4 小時）
   - 企業尾牙活動攝影怎麼找？5 個挑選重點
   - 公司春酒攝影費用行情與注意事項
   - 記者會攝影師需要準備什麼？
   - 請活動攝影師 vs 叫員工拍，差在哪？
   - 台北活動攝影師推薦

3. **服務頁面補強**（半天）

### 第二步：讓人看得懂

4. **活動作品加背景說明**（每篇 10 分鐘）
5. **建立商業作品集獨立頁面**（半天）

### 第三步：讓人敢聯繫

6. **加入合作企業類型**
7. **客戶推薦語**
8. **明確詢價入口**

## 預期成效

- **短期（1 月內）**：修改副標題 + 補強服務頁 → 提升進站訪客轉換
- **中期（3-6 月）**：知識文章帶來搜尋流量
- **長期**：商業攝影關鍵字排名提升，減少對轉介依賴
