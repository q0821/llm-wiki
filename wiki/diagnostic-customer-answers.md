# 顧客管理自我診斷 — 作答記錄

> 來源問卷：[[src-customer-diagnostic-50-questions]]
> 作答日期：2026-04-16

## 一、顧客基本面

| # | 題目 | 回答 |
|---|------|------|
| 1 | 你目前總共有多少個客戶？ | 大約 80 個 |
| 2 | 這些客戶裡面，B2B（盤商、餐廳、超市）佔幾個？B2C（末端消費者）佔幾個？ | 幾乎都是 B2B |
| 3 | 你叫得出你前 10 大客戶的名字嗎？ | 前 5 大可以，有公司報表可查，但數據不即時 |
| 4 | 你知道你的客戶分佈在哪些縣市嗎？ | 台北、新北較多 |
| 5 | 你的客戶是怎麼找到你的？是業務開發、轉介紹、還是客人自己來的？ | 廠商及客戶轉介紹最多，少部分自己來 |
| 6 | 過去一年，你新增了多少個客戶？ | 約 5 個 |
| 7 | 過去一年，你流失了多少個客戶？ | 不確定（資料顯示歷年留存率約 41～48%，每年流失約 30～65 位） |
| 8 | 流失的客戶，你知道他們為什麼不買了嗎？ | 客戶公司轉型或網站更新 |
| 9 | 你有沒有一份完整的客戶名單？還是散在業務的手機裡？ | 有完整名單 |
| 10 | 如果你的業務明天離職，客戶資料會不會跟著消失？ | 成交記錄有保留，洽談中的可能遺失 |

## 二、回購與消費行為

| # | 題目 | 回答 |
|---|------|------|
| 11 | 你的客戶一年平均回購幾次？ | 約 1 次（網頁設計公司，主要是主機費、維護費年度續約） |
| 12 | 你的整體回購率是多少？（買過兩次以上的客戶比例） | 8 成以上 |
| 13 | 客人從第一次買到第二次買，平均間隔多久？ | 11 個月到 1 年 |
| 14 | 有沒有客人買過一次就再也沒出現了？這個比例有多高？ | 有，猜測約 2 成 |
| 15 | 每個客戶平均一次買多少量（金額或數量）？ | 6～10 萬 |
| 16 | 你的客戶下單頻率有沒有在變化？是越買越多還是越買越少？ | 不知道（年度成交金額：2020 最高 610 萬，2021 降至 431 萬，2022 降至 250 萬）|
| 17 | 客人通常都在什麼時間點下單？有沒有季節性？ | 無季節性 |
| 18 | 客人最常買的品項是什麼？最不常買的是什麼？ | 最常：網頁設計服務、主機代管；最不常：網站維護服務 |
| 19 | 有沒有客人只買某一支產品、從來不買其他的？ | 有 |
| 20 | 你的客人有沒有「組合購買」的習慣？（例如 A 產品＋B 產品一起買） | 有 |

## 三、顧客分級

| # | 題目 | 回答 |
|---|------|------|
| 21 | 你有沒有把客戶分級？怎麼分的？ | 有，按照交易金額分級 |
| 22 | 你的前 20% 客戶，貢獻了多少比例的營收？ | 70.88% |
| 23 | 你的 A 級客戶有幾個？他們一年買多少？ | 5 個（格博、原民台、宸曜、永豐證券、ABV），2025 年合計 208.9 萬，平均 41.8 萬／人 |
| 24 | 你的 C 級客戶有幾個？值不值得花時間維護？ | 79 個（資料），平均年消費僅 2.1 萬，不太值得維護 |
| 25 | A 級客戶跟 C 級客戶，你對待方式有什麼不同？ | A 級盡力達成需求；C 級看狀況，做最小時間維護 |
| 26 | 有沒有客戶本來是 C 咖，後來變成 A 咖的？你知道為什麼嗎？ | 沒有印象 |
| 27 | 有沒有客戶本來是 A 咖，後來變成 C 咖甚至消失的？你知道為什麼嗎？ | 不知道 |
| 28 | 你有沒有針對不同等級的客戶，給不同的價格或優惠？ | 有不同價格及優惠 |
| 29 | 你的 A 級客戶如果明天被競爭對手搶走，你有備案嗎？ | 沒有 |
| 30 | 你有沒有定期檢視客戶等級的變動？多久看一次？ | 一年看一次 |

## 四、顧客終身價值 CLV

| 32 | 你的 A 級客戶平均終身價值是多少？ | 不知道（資料估算：歷史平均累計約 58.6 萬） |
| 33 | 你的 B 級客戶平均終身價值是多少？ | 不知道（資料估算：歷史平均累計約 12.3 萬） |
| 34 | 你的 C 級客戶平均終身價值是多少？ | 不知道（資料估算：歷史平均累計約 7.2 萬） |
| 35 | 你獲得一個新客戶的成本是多少？（業務拜訪、樣品、時間成本） | 不知道 |
| 36 | 獲客成本跟客戶終身價值比起來，划算嗎？ | 應該是划算的 |
| 37 | 你有沒有算過，讓一個客人多回購一次，比開發一個新客人便宜多少？ | 沒有 |
| 38 | 如果一個客戶流失了，你損失的不只是一張訂單，而是他未來幾年的訂單，你有算過這個數字嗎？ | 沒有 |

## 五、顧客經營與關係管理

| # | 題目 | 回答 |
|---|------|------|
| 39 | 客人下單之後，你有沒有主動聯繫過他？ | 很少，但有 |
| 40 | 你上一次打電話給你的 A 級客戶是什麼時候？ | 不是自己負責的 |
| 41 | 客人如果三個月沒下單，你有沒有機制去追蹤跟關心？ | 沒有，但因為是年約模式，三個月未下單屬正常 |

| 45 | 你有沒有在重要節日或客人的關鍵時刻做過什麼？（過年送禮、開店祝賀） | 以前有中秋送禮，現在已停止 |
| 46 | 你的客人會不會主動幫你轉介紹？如果不會，你知道為什麼嗎？ | 會 |
| 47 | 你有沒有問過客人：「你為什麼選我們而不是選別人？」 | 有 |
| 48 | 你有沒有問過客人：「我們還可以幫你什麼？」 | 很少 |
| 49 | 你現在的客戶經營，是靠系統在跑，還是靠人的記憶在撐？ | 靠人的記憶 |
| 50 | 如果你今天開始認真做 CRM，你覺得第一步應該做什麼？ | 先列出客戶名單 |

- 答得出來的題數：約 34 題

## 主要盲區

| 類別 | 盲區 |
|------|------|
| 顧客基本面 | 流失客戶數量不清楚，洽談中資料有遺失風險 |
| 回購行為 | 無法追蹤客戶下單頻率的變化趨勢 |
| 顧客分級 | 不清楚等級升降原因；A 級客戶無流失備案 |
| CLV | 各級終身價值、獲客成本、留客 vs 開發成本均未計算 |
| 關係管理 | 客訴未記錄；主動聯繫頻率低；節日經營已停止 |

## 優勢盤點

- 回購率高（8 成以上），年約模式穩定
- 已有客戶分級機制（按交易金額）
- 已有完整客戶名單與成交記錄
- 差異化對待 A／C 級客戶
- 客人會主動轉介紹，且有問過選擇原因
