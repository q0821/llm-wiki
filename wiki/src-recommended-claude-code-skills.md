# 推薦的 Claude Code Skills 清單

> 個人整理推薦｜作者 Jackie Yeh｜記錄日期：2026-04-21

## 摘要

依 GitHub star 數排序的 8 個值得安裝的 Claude Code skills，涵蓋通用、文件處理、團隊審查、Obsidian、行銷、科研、長期內容等不同面向。全部可用 `npx skills add <repo>` 安裝。

| 排名 | Skill | 作者 | Stars | 適用對象 | 一句話 |
|------|-------|------|-------|---------|--------|
| 1 | [superpowers](https://github.com/obra/superpowers) | Jesse Vincent | 16 萬 | 所有人 | 讓 Claude 做事更嚴謹 |
| 2 | [anthropics/skills](https://github.com/anthropics/skills) | Anthropic 官方 | 12 萬 | 所有人 | 文件處理（Word / PPT / Excel / PDF） |

| 5 | [claude-code-templates](https://github.com/davila7/claude-code-templates) | Daniel Ávila | 2.5 萬 | Skill 新手 | 大型工具集 |

## 挑選觀察

- **Stars 斷層**：前三名（16 萬 / 12 萬 / 7.7 萬）顯著領先，可作為新安裝者的優先順序
- **場景特化**：obsidian-skills（筆記流）、marketingskills（行銷）、claude-scientific-skills（研究）各自鎖定清楚的使用者 profile
- **官方 + 社群**：Anthropic 官方（文件）與 Jesse Vincent（嚴謹度）、Garry Tan（團隊視角）並列，顯示 Claude Code 生態系已經從官方單點擴散到社群多點
