# Agent Skills

## 概念說明

一種讓 AI coding agents（如 Claude Code、Cursor、Windsurf 等）可擴充任務能力的規範。每個 skill 是一份 Markdown 文件，定義特定任務的框架、檢查清單與工作流程。Agent 面對對應任務時會自動套用。

## 規範

- 官方規範：https://agentskills.io
- 安裝方式：`npx skills add <repo>`
